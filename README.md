# Desarrollo App Web

Proyecto web realizado como parte de una actividad académica enfocada en el uso de **HTML y CSS** para la creación de un sitio web estructurado y funcional.


## Descripción

Este proyecto consiste en el desarrollo de un sitio web básico con múltiples páginas, aplicando buenas prácticas de maquetación y estilos con CSS.


## Tecnologías utilizadas

- HTML5
- CSS3


## Estructura del proyecto

Desarrollo_App_Web/
│── INDEX.HTML
│── Contacto.html
│── Registro.html
│── servicios.html
│── acerca de.html
│── CSS/
│   └── estilos.css
│── IMG/


## Funcionalidades

- Navegación entre páginas
- Diseño estructurado con HTML
- Aplicación de estilos con CSS
- Formularios básicos
