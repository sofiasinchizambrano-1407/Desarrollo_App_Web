Bienvenidos

Este sitio web presenta una estructura básica en HTML con estilos externos en CSS, incorporando navegación, contenido informativo, tabla de servicios y formularios.

¿Qué encontrarás aquí?

  • Página de inicio con presentación general.
  • Sección Acerca de con información institucional.
  • Tabla de servicios con marcos y estilos en CSS.
  • Formulario de contacto y formulario de registro.

Características del proyecto

El diseño utiliza etiquetas semánticas como header, nav, main, section, article, figure, table, form y footer.

Todo el formato visual fue organizado desde un archivo CSS externo.
